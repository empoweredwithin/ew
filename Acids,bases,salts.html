C) Question and Answers

  1. Nitric acid is not used in the preparation of hydrogen because it is a strong oxidizing agent and tends to oxidize hydrogen to water, preventing the collection of hydrogen gas.
  2. Adding water to acid releases a lot of heat, which can cause the mixture to splatter and cause burns. Instead, acid should always be added slowly to water to control the exothermic reaction.
  3. A monobasic acid is one that can donate only one proton (H⁺) per molecule. Acetic acid (CH₃COOH) has only one replaceable hydrogen atom, which is why it is called monobasic.
  4. Basicity refers to the number of hydrogen ions (H⁺) an acid can donate per molecule during a reaction. A monobasic acid can donate only one hydrogen ion. Examples: Hydrochloric acid (HCl) and Nitric acid (HNO₃).
  5. i) 2NO₂ + H₂O → HNO₃ + HNO₂
    ii) NaCl + H₂SO₄ → NaHSO₄ + HCl
  6. Dilute hydrochloric acid is stronger because it fully ionizes in water, producing a higher concentration of hydrogen ions (H⁺), while acetic acid only partially ionizes, making it a weak acid.
  7. i) A: 2Fe + 3Cl₂ → 2FeCl₃
    ii) E: Pb(NO₃)₂ + 2HCl → PbCl₂ + 2HNO₃
    iii) D: H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O
    iv) C: CuO + H₂SO₄ → CuSO₄ + H₂O
  8. a) Blue litmus will turn red, indicating an acidic solution.
    b) Red litmus will turn blue, indicating a basic solution.
    c) Red litmus will turn blue, indicating a strong alkaline solution.
  9. Zinc sulphate - Neutralization:
    ZnO + H₂SO₄ → ZnSO₄ + H₂O

    Ferrous sulphide - Oxidation:
    2FeS + 3O₂ → 2FeO + 2SO₂

    Barium sulphate - Precipitation:
    BaCl₂ + H₂SO₄ → BaSO₄ + 2HCl

    Ferric sulphate - Neutralization:
    2Fe(OH)₃ + 3H₂SO₄ → Fe₂(SO₄)₃ + 6H₂O

    Sodium sulphate - Synthesis:
    2Na + H₂SO₄ → Na₂SO₄ + H₂
  10. a) If M is a metal, then the litmus will turn blue.
    b) If M is a non-metal, then the litmus will turn red.
    c) If M is a reactive metal, then hydrogen will be evolved when M reacts with dilute sulphuric acid.
    d) If M is a metal, it will form a basic oxide, which will form an alkaline solution with water.
